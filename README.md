# 🐍 Python Learning Path — Beginner to Intermediate in 3 Months

A practical, project-based learning program following the **80% practice / 20% theory** principle.

## Philosophy

You don't learn Python by reading about Python — you learn it by **writing Python, breaking it, and fixing it**.

**Weekly Structure:**
- **20% Theory** (~1-2h): Watch a short video, read docs, understand the "why"
- **80% Practice** (~6-8h): Solve exercises, build mini-projects, reuse what you learned

**Recommended pace:** 8-10h/week spread over 4-5 days. Can compress to 6-8 weeks if you have more time.

**Golden Rule:** Never copy a solution without typing it yourself AND being able to explain every line.
